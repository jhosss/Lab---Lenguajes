Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

    HTML 5 (Hyper Text Markup Language.version 5) es la quinta revision importante del lenguaje básico de la World Wide Web HTML

    Use la extension .html

  • CSS 3

    Hoja de estilo en cascada o CSS (siglas en ingles de cascading style sheet)es un lenguaje usado para definir y crear la presentación de una documento estructurado escrito en HTML.

    Usa la extension .css

  • Javascript

    Javascript (abreviado comunmente JS) es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side) implementado como parte de un navegador web permitiendo mejoras enla interfa de usuario y páginas web dinámicas.

    Usa la extensión .js
